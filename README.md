# Arx Libertatis — HUD Opacity Patch

> **This is a patched fork of [Arx Libertatis 1.2.1](https://github.com/arx/ArxLibertatis), the open-source engine reimplementation of Arx Fatalis.**
> The upstream project lives at **https://github.com/arx/ArxLibertatis**.

---

## What this fork adds

A **HUD opacity slider** in the Options → Interface menu, directly beneath the existing HUD Scale slider.

Moving it fades the entire in-game HUD — health and mana orbs, side icons (backpack, book, torch, purse, steal, level-up, change-level), stealth gauge, damaged-equipment indicators, spell slots, screen-edge arrows, hit-strength gauge, memorized rune display, and gold number — all the way from fully opaque (default) down to invisible. Menus, loading screens, cinematics, and the player book interior are unaffected.

The setting persists to `cfg.ini` as `hud_opacity` under the `[interface]` section and defaults to `1.0` (fully opaque, identical to unpatched behavior).

## The patch

**5 files changed, 64 insertions, 37 deletions:**

| File | Change |
|---|---|
| `src/core/Config.h` | Added `float hudOpacity` to the interface config struct |
| `src/core/Config.cpp` | Default `1.0`, config key `hud_opacity`, writer and clamped reader |
| `src/gui/MainMenu.cpp` | Slider widget on the Interface options page, `onChangedHudOpacity` callback |
| `src/gui/Hud.cpp` | `hudAlpha()` helper; `OpModulate` texture stage bracket in `HudRoot::draw()`; 15 draw-call modifications |
| `src/gui/hud/HudCommon.cpp` | Opacity applied in `HudIconBase::draw()` for all icon base-class elements |

**Technical detail:** Arx's OpenGL texture stage defaults to `GL_COMBINE_ALPHA = GL_REPLACE`, which discards vertex alpha entirely. The fix switches to `GL_MODULATE` (texture alpha × vertex alpha) for the duration of `HudRoot::draw()` only, then restores the default. This means scaling vertex alpha by `hud_opacity` works uniformly for both standard-blend and additive-blend HUD elements.

A standalone patch file is included as `arx-hud-opacity.patch` and can be applied to a clean 1.2.1 tree:

```bash
cd ArxLibertatis-1.2.1
patch -p1 < arx-hud-opacity.patch
```

## Building

Dependencies (Arch / CachyOS):
```bash
sudo pacman -S cmake ninja boost glm glew epoxy libpng bzip2 zlib freetype2 openal sdl2
```

Configure and build:
```bash
mkdir build && cd build
cmake ../ArxLibertatis-1.2.1 -DCMAKE_BUILD_TYPE=RelWithDebInfo -G Ninja
ninja arx
```

Run (game data auto-discovered at `~/.local/share/arx/`):
```bash
./build/arx
```
